🎯 CSS Position Property - Examples

1. Position: Static (Default)

Static default position hai. Elements normal document flow mein rehte hain. Top, left, right, bottom properties kaam nahi karti.

position: static;
Box 1 (Static)
Box 2 (Static)
Box 3 (Static)

2. Position: Relative

Relative position mein element apni original position ke relative move hota hai. Top, left, right, bottom properties use kar sakte hain.

position: relative;
top: 20px;
left: 30px;
Box 1 (Normal)
Box 2 (Moved: top:20px, left:30px)
Box 3 (Normal)

3. Position: Absolute

Absolute position mein element nearest positioned parent (relative/absolute/fixed) ke relative position leta hai. Agar koi positioned parent nahi hai, to document body ke relative position leta hai.

position: absolute;
top: 10px;
right: 10px;
Top Left
Top Right
Bottom Left
Bottom Right

Ye container relative position par hai, isliye absolute boxes iske relative position le rahe hain.

4. Position: Fixed

Fixed position mein element viewport ke relative position leta hai. Page scroll karne par bhi element same jagah rehta hai. Navigation bars aur chat buttons ke liye perfect hai.

position: fixed;
top: 20px;
right: 20px;
🔝 Fixed Top Right
(Scroll karo - ye rahega yahi)
🔽 Fixed Bottom Right
(Scroll karo - ye bhi rahega yahi)

Page scroll karo - fixed boxes hamesha same jagah rahenge!

5. Position: Sticky

Sticky position relative aur fixed ka combination hai. Element tab tak relative rehta hai jab tak scroll nahi hota, phir fixed ho jata hai jab tak uska container visible hai.

position: sticky;
top: 0;

📝 Summary

  • Static: Default position, normal flow
  • Relative: Apni jagah se relative move karta hai
  • Absolute: Nearest positioned parent ke relative
  • Fixed: Viewport ke relative, scroll par bhi same jagah
  • Sticky: Relative se Fixed tak, scroll ke basis par
